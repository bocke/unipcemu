horizontally:
first half:

position 0 start for the first half = ((horizontal total+5)*dots/character) - (end vertical retrace + start vertical retrace)

0 - display enable skew: left overline
display enable skew - end horizontal display: active display
end horizontal display - start horizontal blanking:  right overline
start horizontal blanking - start vertical retrace: black right border

Off-screen (second half, =left side of the screen):
start vertical retrace - (start vertical retrace + end vertical retrace): do nothing, except reset screen x to zero.
(start vertical retrace + end vertical retrace) - (start horizontal blanking + end horizontal blanking): left clear till extra border.
(start horizontal blanking + end horizontal blanking) - ((horizontal total+5)*dots/character): first overline.

vertically:
first half:
0 - vertical display end: active display
vertical display end - vertical blank start: overline
vertical blank start - vertical retrace start: black bottom border

Off-screen (second half, =top side of the screen)
vertical retrace start - (vertical retrace start + vertical retrace end): do nothing, except reset y to zero.
(vertical retrace start + vertical retrace end) - (vertical blank start + vertical blank end): black top border
(vertical blank start + vertical blank end) - vertical total: top overline.
